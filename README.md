# Personal Portfolio Website

This repository contains the source code for a responsive personal portfolio website designed for Bayuzen Ahmad, an AI Engineer. The website showcases professional skills, projects, and contact information with a modern, dark-themed aesthetic.

## Features

-   **Responsive Design**: The layout adapts seamlessly to desktops, tablets, and mobile devices.
-   **Interactive Elements**: Includes typing text animations, smooth scrolling, and hover effects on cards and buttons.
-   **Downloadable CV**: A direct link to download the professional curriculum vitae.
-   **Work Experience Timeline**: A detailed list of professional roles and companies.

## Technologies Used

-   **HTML5**: For semantic structure.
-   **CSS3**: For styling, including Flexbox, Grid, and animations.
-   **JavaScript**: For interactive components like the mobile navigation menu and typing effects.
-   **Typed.js**: A JavaScript library for the typing animation effect.
-   **Boxicons**: For vector icons used throughout the site.
-   **ScrollReveal**: For scroll animations.

## Installation and Usage

To view the website locally:

1.  Clone this repository to your local machine.
2.  Navigate to the project directory.
3.  Open the `index.html` file in any modern web browser.

## Project Structure

-   `index.html`: The main HTML document.
-   `css/style.css`: The stylesheet containing all visual styles and responsive rules.
-   `js/script.js`: The JavaScript file for handling UI interactions.
-   `images/`: Directory containing project assets.
-   `cv/`: Directory containing the downloadable resume.
